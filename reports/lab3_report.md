## 2. Процесс деплоя

### Пошаговая инструкция

1. **Подготовка репозитория**
   - Убедился, что `.env` в `.gitignore` (токен не попадёт в git)
   - Проверил, что `requirements.txt` содержит все зависимости
   - Добавил файл `Procfile`:
     ```
     worker: python bot.py
     ```

2. **Загрузка на GitHub**
   ```bash
   git add .
   git commit -m "Prepare for deploy"
   git push origin main
   ```

3. **Настройка Railway**
   - Зарегистрировался на [railway.app](https://railway.app) через GitHub
   - Создал новый проект → «Deploy from GitHub repo»
   - Подключил репозиторий `fedor-vibecoding`

4. **Переменные окружения**
   - В настройках проекта на Railway добавил переменную `BOT_TOKEN`
   - Вставил токен бота из BotFather

5. **Деплой**
   - Railway автоматически определил Python-проект
   - Установил зависимости из `requirements.txt`
   - Запустил бота

### С какими проблемами столкнулся

| Проблема | Решение |
|---|---|
| Railway не находил точку входа | Добавил `Procfile` с указанием `worker: python bot.py` |
| Бот использовал polling, а Railway ожидал HTTP-сервер | Тип сервиса изменён на `worker` (не `web`) — polling работает без HTTP |
| Файл `favorites.db` пропадал после перезапуска | Это ожидаемое поведение для бесплатного тарифа; для продакшена нужен внешний PostgreSQL |

## 5. Улучшения на основе фидбека

На основе собранных отзывов были реализованы **3 улучшения**:

### Улучшение 1: Сравнение погоды в двух городах

**Проблема:** Алексей и Игорь хотели сравнивать погоду, чтобы выбирать куда поехать.

**Решение:** Добавлена команда `/compare Город1, Город2` и кнопка «⚖️ Сравнить города» в меню.

Бот показывает погоду обоих городов рядом и подводит итог: где теплее, суше и тише.

> 📸 *[Вставить скриншот: результат команды /compare Москва, Сочи]*

### Улучшение 2: UV-индекс и время рассвета/заката

**Проблема:** Мария хотела видеть UV-индекс для защиты кожи и время рассвета/заката для планирования прогулок.

**Решение:** В текущую погоду добавлены:
- UV-индекс с интерпретацией (низкий / средний / высокий + рекомендация по крему SPF)
- Время восхода и заката

> 📸 *[Вставить скриншот: текущая погода с UV-индексом и восходом/закатом]*

### Улучшение 3: Советы по одежде в прогнозе

**Проблема:** Дмитрий заметил, что в прогнозе на 3 дня нет советов по одежде, хотя в текущей погоде они есть.

**Решение:** В каждый день прогноза добавлена краткая рекомендация по одежде (основная, без подробностей об осадках и ветре — чтобы не перегружать прогноз).
