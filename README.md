# Погодный Советник — Telegram-бот

Telegram-бот, который показывает погоду и даёт рекомендации: что надеть и чем заняться.

Проект выполнен в рамках курса «Vibe Coding: AI-боты для бизнеса» (ИТМО, 2025/2026).

## Возможности

- **Текущая погода** по любому городу мира
- **Прогноз на 3 дня** с подробностями и советами по одежде
- **Советы по одежде** — зонт, куртка, шорты в зависимости от погоды
- **Рекомендации по активностям** — пробежка, кино, пикник...
- **Сравнение городов** — где теплее, суше и тише
- **UV-индекс** и **время рассвета/заката**
- **Избранные города** — сохраняй и получай погоду в один клик

## Стек технологий

- Python 3.10+
- [python-telegram-bot](https://python-telegram-bot.org/) — работа с Telegram API
- [Open-Meteo API](https://open-meteo.com/) — погодные данные (бесплатно, без ключа)
- SQLite — хранение избранных городов пользователей
- Docker — контейнеризация для деплоя

## Установка и запуск

### Вариант 1: Локальный запуск

```bash
# Клонируйте репозиторий
git clone <url-репозитория>
cd fedor-vibecoding

# Создайте виртуальное окружение
python3 -m venv .venv
source .venv/bin/activate

# Установите зависимости
pip install -r requirements.txt

# Настройте токен бота
cp .env.example .env
# Откройте .env и вставьте токен из @BotFather

# Запустите бота
python bot.py
```

### Вариант 2: Docker

```bash
# Настройте токен
cp .env.example .env
# Откройте .env и вставьте токен

# Запустите
docker-compose up -d --build

# Логи
docker-compose logs -f bot

# Остановка
docker-compose down
```

### Вариант 3: Railway

1. Загрузите код на GitHub
2. Зарегистрируйтесь на [railway.app](https://railway.app)
3. Создайте проект из GitHub-репозитория
4. Добавьте переменную `BOT_TOKEN` в настройках
5. Бот автоматически запустится

## Команды бота

| Команда | Описание |
|---------|----------|
| `/start` | Приветствие и главное меню |
| `/help` | Справка по командам |
| `/forecast Город` | Прогноз на 3 дня |
| `/compare Город1, Город2` | Сравнить погоду в двух городах |
| `/save Город` | Добавить город в избранное |
| `/favorites` | Список избранных городов |
| `/remove Город` | Удалить город из избранного |

Также можно просто написать название города — бот покажет текущую погоду с рекомендациями.
